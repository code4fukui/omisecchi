# omisecchi

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

[そだてて！おみせっち](https://code4fukui.github.io/omisecchi/) by 北陸観光ハッカソン チームD

A character-raising game concept developed for the Hokuriku Tourism Hackathon. This project serves as a simple, interactive prototype.

## Demo & Project Page

-   **Live Demo:** [https://code4fukui.github.io/omisecchi/](https://code4fukui.github.io/omisecchi/)
-   **Team D Hackathon Entry:** [fukuno.jig.jp/4512](https://fukuno.jig.jp/4512)

## Features

-   **Interactive Prototype:** A simple landing page (`index.html`) that leads to a main application screen (`main.html`).
-   **Mocked-up UI:** The main screen features a fixed header and a clickable bottom navigation bar.
-   **Concept Wireframes:** The "Skill" section displays a hand-drawn wireframe (`skill.jpg`) showing a potential UI design.
-   **Placeholder Functionality:** The "Mission" and "Test" buttons use JavaScript `alert()`s to demonstrate intended features.
-   **External Content:** The "Friend" and "Mail" sections embed content from the Hokuriku Gift Campaign website using iframes.

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in a web browser.
3.  Click the "そだてはじめる！" (Start Raising!) button.
4.  Use the bottom navigation bar to explore the different sections.
