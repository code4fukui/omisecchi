# omisecchi

[そだてて！おみせっち](https://code4fukui.github.io/omisecchi/) by 北陸観光ハッカソン チームD

北陸観光ハッカソン向けに開発されたキャラクター育成ゲームのコンセプトです。本プロジェクトは、シンプルでインタラクティブなプロトタイプとして機能します。

## デモ & プロジェクトページ

- **ライブデモ:** [https://code4fukui.github.io/omisecchi/](https://code4fukui.github.io/omisecchi/)
- **チームD ハッカソンエントリー:** [fukuno.jig.jp/4512](https://fukuno.jig.jp/4512)

## 機能

- **インタラクティブなプロトタイプ:** シンプルなランディングページ（`index.html`）からメインアプリケーション画面（`main.html`）へと遷移します。
- **モックアップUI:** メイン画面には、固定ヘッダーとクリック可能な下部ナビゲーションバーが実装されています。
- **コンセプトワイヤーフレーム:** 「スキル」セクションでは、手書きのワイヤーフレーム（`skill.jpg`）を表示し、想定されるUIデザインを提示します。
- **プレースホルダ機能:** 「ミッション」および「テスト」ボタンは、JavaScriptの `alert()` を使用して想定される機能のデモを行います。
- **外部コンテンツ:** 「フレンド」および「メール」セクションでは、iframeを使用して北陸ギフトキャンペーンサイトのコンテンツを埋め込んでいます。

## はじめに

1. リポジトリをクローンします。
2. Webブラウザで `index.html` を開きます。
3. 「そだてはじめる！」ボタンをクリックします。
4. 下部ナビゲーションバーを使用して、各セクションを操作します。
